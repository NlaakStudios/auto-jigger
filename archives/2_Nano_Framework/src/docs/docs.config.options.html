Options

  • fullscreen

    • Type: Boolean

    • True - run in fullscreen mode.
    • False - use can choose fullscreen manually

  • orientation

    • Type: String

    • Auto - App may run in either mode.
    • Portrait - App is froced into Portrait mode.
    • Landscape - App is forced into Landscape mode.

  • analytics

  • htmlcachectrl

    • Type: String

    • Public - May be cached in public shared caches.
    • Private - May only be cached in private cache.
    • no-Cache - May not be cached.
    • no-Store - May be cached but not archived.
