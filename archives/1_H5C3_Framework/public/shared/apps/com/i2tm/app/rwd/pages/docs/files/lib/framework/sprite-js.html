h5c3.Sprite

[Extends <a href=’h5c3.Sprite’>h5c3.Pooled</a>] <p> Sprites are instances of a sprite sheet used for rendering.  <p> To create a sprite, pass a sprite sheet into the constructor: <pre><code> var zombieSprite = new h5c3.Sprite( zombieSpriteSheet ); </code></pre> You can then use setAnimation to select an animation from the sheet: <pre><code> zombieSprite.setAnimation(‘attacking right’); </code></pre> To draw the sprite, use the draw method: <pre><code> zombieSprite.draw(h5c3.device.ctxGame, 100, 100); </code></pre> To cycle animations, call update: <pre><code> zombieSprite.update(h5c3.device.elapsed); </code></pre> <p> Check the <a href=’url target=”http://playcraftlabs.com/develop/guide/spritesandanimation” name=”http://playcraftlabs.com/develop/guide/spritesandanimation”’>sprites and animation guide</a> for more information and features.
