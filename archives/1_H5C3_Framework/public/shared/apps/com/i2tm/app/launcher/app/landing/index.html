H5C3 Framework

Version 0.9.6

H5C3 is a HTML5 & CSS3 framework for creating next generation web applications. It includes an Engine, Plugins and Responsive Sytlesheets. It will run on any device that can run Google Chrome. I have no intention at this time of supporting the other browsers. They need to comply with the W3C standards. However, there is no need because Chrome runs on just about every platform, with a little tweaking you can probably get it installed on your smart refrigerator!
