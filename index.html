Auto Jigger

A simple tool for web developers that automatically handles the inclusion of Modernizr, jQuery, Bootstrap and Font-Awesome as well as any other third party library or framework into web page.

About

What is is Not...

First off autojigger is not meant to replace other Frameworks.

What is is...

It is extremely fast and lightweight. Think of it more as a starting point or boilerplate for front end cross-platform responsive web development. It is written and pure JavaScript and uses his enclosures (100% Advanced Optimations Compliant). It uses HTML5, CSS3 and of course JavaScript.

It's Purpose...

It's primary task it's too load you're Frameworks, libraries and local content very easily.

Helper Functions

lds

Utility function for loading JavaScript or Stylesheets. No other file formats are supported.

Parameters:

  • @param {string} $fileType either script or link
  • @param {string} $id element ID of this file
  • @param {string} $url full path to file, local or remote
  • @param {boolean} $tail set to true if you want script loaded into tail.
  • @param {string} $after make sure loaded file is after this given file
  • @param {function} $onLoad optional
  • @param {function} $onError optional
Returns: Boolean - True is loaded, False otherwise.
Example:

						las([
							'https://cdnjs.cloudflare.com/ajax/libs/react/15.3.1/react.js',
							'https://cdnjs.cloudflare.com/ajax/libs/react/15.3.1/react-dom.js',
							'assets/css/reactjs_todo_app.css',
							'assets/js/reactjs_todo_app.js'
						]);

lds

Used to load a list of files.

Parameters:

  • @param {Array} files all scripts to be loaded.
Returns: N/A
Example:

						W$['lds'](
							'js', 
							'jquery', 
							'https://code.jquery.com/jquery-2.2.4.min.js', 
							true, 
							'last', 
							function(){
								W$['las']([
									'https://cdnjs.cloudflare.com/ajax/libs/modernizr/2.8.3/modernizr.min.js',
									'https://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/css/bootstrap.min.css',
									'https://maxcdn.bootstrapcdn.com/font-awesome/4.6.3/css/font-awesome.min.css',
									'https://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/js/bootstrap.min.js'
								])
							}, 
							null
						);

vld

Check if a value is valid (not null, undefined or empty)

Parameters:

  • @param {*} a A value
Returns: Boolean
Example:

						if !(vld(foo)) { someFunction(); }

chk

Checks if a param is valid (null or undefined) in which case the default value will be returned

Parameters:

  • @param {*} $p Parameter to check
  • @param {*} $def Default value to return if p is either null or undefined
Returns: Mixed - if valid, otherwise def (default)
Example:

						if (foo=chk(foo,"bar")) { someFunction(); }

gei

Alias for window.document.getElementById

Parameters:

  • @param {string} $p
Returns: DOMElement
Example:

						var element = gei("myDIV");

cls

Set, add, remove or replace an elements classes. the second class $c2 is only needed when using the replace action.

Parameters:

  • @param {string} $el element id
  • @param {string} $action action [set|clear|add|remove|replace|toggle]
  • @param {string} $c1 name of class to work with
  • @param {string} $c2 name of class used in replace of $c1
Returns: N/A
Example:

						cls('wrapper','add','vanish','');

gex

Returns file extension from a url string

Parameters:

  • @param {String} a
Returns: String - File Extension
Example:

						d=location.pathname + 'html/' + gfn($url) + '.inc' + gex($url);

gfn

Returns file name from a url string

Parameters:

  • @param {String} a
Returns: String - Filename without extension
Example:

						d=location.pathname + 'html/' + gfn($url) + '.inc' + gex($url);
